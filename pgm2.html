LISTS IN HTML


ORDERED LIST

An ordered list in HTML is used to display a list of items in a specific, numbered order. It is created using the 'ol' tag, and each list item is defined with the 'li' tag. Ordered lists are useful when the order of items matters, such as in instructions, rankings, or steps in a process. By default, the browser numbers each item starting from 1, but this can be customized using attributes.

  1. Decimal (Numbers)
  2. Uppercase Letters
  3. Lowercase Letters
  4. Uppercase Roman Numerals
  5. Lowercase Roman Numerals

UNORDERED LIST

An unordered list in HTML is used to group a set of related items in no specific order. It is created using the 'ul' tag, and each item in the list is defined with the 'li' (list item) tag. Unlike ordered lists, unordered lists do not use numbers. Instead, they use bullets (‚óè) or other markers like circles or squares, depending on the browser or list style. Unordered lists are commonly used for things like menus, features, shopping lists, or any content where the order of items doesn't matter.

  • Disc
  • Circle
  • Square
  • None

DEFINITION LIST

A definition list in HTML is used to display terms and their corresponding descriptions, making it ideal for glossaries, dictionaries, or any content that requires a term-definition format. It consists of three main tags: 'dl' for the definition list container, 'dt' for each term (definition term), and 'dd' for the description (definition description). Unlike ordered or unordered lists, a definition list does not use bullets or numbers. Instead, it pairs each term with its explanation in a structured and readable way. This type of list is especially useful for presenting information clearly and semantically in educational or reference content.
